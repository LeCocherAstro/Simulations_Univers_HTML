**Auteur / Author : Pierre R. — Le Cocher**

---

## 🇫🇷 Français

### Présentation

Ce dépôt rassemble plusieurs simulations autonomes de mécanique gravitationnelle et de dynamique stellaire. Chaque fichier `.html` est complet : il suffit de l'ouvrir dans un navigateur récent pour lancer la simulation. Le rendu s'appuie sur le `<canvas>` et, pour les scènes 3D, sur **WebGL** (via **Three.js**).

### Contenu

| Fichier | Description |
|---------|-------------|
| [gravity2.html](gravity2.html) | **Puits de gravité** — simulation N-corps en 3D temps réel (WebGL / Three.js). Intégration gravitationnelle et caméra orbitale. |
| [cluster-3k.html](cluster-3k.html) | **Amas globulaire** — distribution stellaire suivant le modèle de Plummer (palier 1). |
| [nebula.html](nebula.html) | **Nébuleuse** — visualisation de la naissance d'un amas stellaire. |
| [opencluster.html](opencluster.html) | **Amas ouvert** — séquence principale et point de *turn-off* du diagramme HR. |

### Utilisation

1. Cloner ou télécharger le dépôt.
2. Ouvrir le fichier `.html` souhaité dans un navigateur moderne (Chrome, Firefox, Safari, Edge).

> 💡 Aucune dépendance à installer ni serveur à démarrer. Les fichiers fonctionnent en local par simple double-clic.

### Prérequis

- Un navigateur compatible **WebGL** pour les scènes 3D.
- Une machine récente est recommandée pour les simulations comportant un grand nombre de corps.

## 🇬🇧 English

### Overview

This repository gathers several self-contained simulations of gravitational mechanics and stellar dynamics. Each `.html` file is standalone: simply open it in a recent browser to run the simulation. Rendering relies on `<canvas>` and, for 3D scenes, on **WebGL** (via **Three.js**).

### Contents

| File | Description |
|------|-------------|
| [gravity2.html](gravity2.html) | **Gravity well** — real-time 3D N-body simulation (WebGL / Three.js). Gravitational integration and orbital camera. |
| [cluster-3k.html](cluster-3k.html) | **Globular cluster** — stellar distribution following the Plummer model (stage 1). |
| [nebula.html](nebula.html) | **Nebula** — visualization of the birth of a star cluster. |
| [opencluster.html](opencluster.html) | **Open cluster** — main sequence and *turn-off* point of the HR diagram. |

### Usage

1. Clone or download the repository.
2. Open the desired `.html` file in a modern browser (Chrome, Firefox, Safari, Edge).

> 💡 No dependencies to install and no server to start. The files work locally with a simple double-click.

### Requirements

- A **WebGL**-capable browser for the 3D scenes.
- A recent machine is recommended for simulations involving a large number of bodies.
